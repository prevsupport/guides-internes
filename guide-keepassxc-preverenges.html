Guide KeePassXC

Installation, configuration et utilisation du gestionnaire de mots de passe

01

Pourquoi un gestionnaire de mots de passe

Aujourd’hui : KeePassXC, un gestionnaire de mots de passe.

Le risque numéro 1 : réutiliser le même mot de passe partout. Un site piraté, et tous vos comptes tombent.

KeePassXC crée un mot de passe unique pour chaque site. Vous ne retenez plus qu’un seul mot de passe : celui qui protège le coffre.

Gratuit, et tout reste sur votre poste, pas sur un cloud.

Si vous êtes employé de l’administration, l’étape de téléchargement et d’installation est déjà réalisée par le référent informatique : vous pouvez passer directement à l’étape 04.
02

Téléchargement

Rendez-vous sur le site officiel keepassxc.org, onglet Windows.

Cliquez sur Download for Windows.

03

Installation

Lancez le fichier téléchargé, cliquez sur Suivant, Suivant, Terminer.

Au premier lancement, on passe l’interface en français : Réglages, Général, Langue.

04

Créer une nouvelle base de données

On clique sur Créer une base de données.

On lui donne un nom, par exemple « MotsDePasse_Préverenges ». La description est facultative. On clique sur Continuer.

05

Paramètres de chiffrement

Ici, ce sont les réglages de sécurité interne. On garde les valeurs par défaut, elles conviennent très bien. On clique sur Continuer.

06

Le mot de passe maître

On crée le mot de passe maître, celui qui protège toute la base. On le choisit fort et unique. On confirme, puis on clique sur Terminer.

Attention : si ce mot de passe est perdu, la base est irrécupérable.
07

Où enregistrer le fichier

On choisit où enregistrer le fichier. Deux options : soit on le garde sur la machine, dans un dossier précis et facile à retrouver, soit on le met sur un drive pour y accéder depuis plusieurs postes.

Si vous le mettez sur un drive, il est fortement recommandé de chiffrer le fichier avec un outil comme Cryptomator — mais ça, ce sera l’objet d’un autre guide.

08

Découvrir l’interface

Voici l’interface. À gauche, les groupes ou dossiers pour organiser vos entrées. Au centre, la liste des entrées du groupe sélectionné. En haut, la barre de recherche pour retrouver rapidement un mot de passe.

09

Créer une entrée

Clic droit, Nouvelle entrée. On remplit le titre du site, le nom d’utilisateur, le mot de passe, et l’URL. On clique sur OK.

10

Générer un mot de passe

On clique sur l’icône dé pour générer un mot de passe. On choisit la longueur et les types de caractères, puis on clique sur Appliquer le mot de passe.

11

Organiser ses entrées en groupes

Clic droit sur Racine, Nouveau groupe. On peut créer des dossiers comme Communal, Perso, Applications web, pour ranger ses entrées.

12

Joindre un fichier à une entrée

Dans l’onglet Avancé, on peut joindre un fichier à une entrée, par exemple un document de récupération, via Ajouter un fichier puis Charger du disque.

13

Configurer un code à double authentification (TOTP)

Clic droit sur l’entrée, Mot de passe dynamique TOTP, Configurer le TOTP. On colle la clé secrète fournie par le site, on garde les paramètres par défaut, puis OK.

Un code à 6 chiffres se génère automatiquement et se copie en un clic.

14

Sécurité au quotidien

Dans les Paramètres, onglet Sécurité : on retrouve le délai avant vidage automatique du presse-papier — ici 10 secondes après avoir copié un mot de passe — et le verrouillage automatique de la base après une inactivité, ici 15 minutes.

La base se verrouille aussi automatiquement si l’écran se verrouille.

15

Activer l’intégration au navigateur

Dans Paramètres, Intégration aux navigateurs, on active l’option, puis on coche le navigateur utilisé — ici Edge. On clique sur le lien pour télécharger l’extension KeePassXC-Browser.

16

Connecter le navigateur à la base

Dans le navigateur, on clique sur l’icône de l’extension, puis Se connecter. KeePassXC demande de confirmer le jumelage : on donne un nom à cette connexion, par exemple le nom du navigateur, puis Enregistrer et permettre l’accès.
