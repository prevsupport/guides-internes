Guide kDrive

Accès, partage et gestion des fichiers sur kDrive Infomaniak

🏠

1. Écran d'accueil de kDrive

Cette première vue présente les principales rubriques disponibles dans kDrive ainsi que leur utilité.

Dossiers de l'organisation

Les Dossiers de l'organisation sont destinés au stockage des documents appartenant à l'administration. Tous les fichiers et dossiers créés ou modifiés dans cette partie restent la propriété de l'organisation.

Même lorsqu'un collaborateur quitte l'administration, les documents auxquels il a contribué sont conservés et restent accessibles aux personnes disposant des autorisations nécessaires.

À privilégier pour tous les documents de travail, les dossiers d'équipe et les fichiers qui doivent être conservés par l'administration.

Mon dossier

Mon dossier est un espace personnel propre à chaque collaborateur.

Il est possible d'y stocker des fichiers privés ou temporaires et de les partager avec d'autres utilisateurs si nécessaire. En revanche, lorsque le collaborateur quitte l'administration, l'intégralité du contenu de Mon dossier est supprimée, y compris les éventuels partages.

À utiliser uniquement pour les documents personnels ou temporaires qui n'ont pas vocation à être conservés par l'administration.

Mes favoris

Cette rubrique permet d'accéder rapidement aux fichiers et dossiers marqués comme favoris.

Dernières modifications

Affiche les derniers fichiers consultés ou modifiés afin de retrouver facilement les documents récemment utilisés.

Partages

La section Partages regroupe tous les éléments liés aux partages de fichiers. Elle contient plusieurs sous-rubriques :

  • Partagés avec moi : documents ou dossiers que d'autres utilisateurs ont partagés avec vous.
  • Mes partages : documents ou dossiers que vous avez partagés avec d'autres personnes.
  • Liens de partage public : liste de tous les liens publics que vous avez créés. Cette rubrique permet de contrôler les liens existants et de les supprimer lorsqu'ils ne sont plus nécessaires.

Boîtes de dépôt

Cette rubrique regroupe toutes les boîtes de dépôt créées.

Une boîte de dépôt permet à des personnes de déposer des fichiers sans avoir accès au contenu du dossier. Elle est notamment utilisée pour la collecte de documents, de formulaires ou de pièces justificatives. Son fonctionnement sera présenté dans une section dédiée.

Corbeille

La Corbeille contient les fichiers et dossiers supprimés. Tant qu'ils y sont présents, ils peuvent être restaurés si nécessaire, selon les règles de conservation définies par l'organisation.

🖱️

2. Menu contextuel (clic droit)

Lorsqu'un clic droit est effectué dans un dossier ou sur un fichier, un menu contextuel apparaît avec plusieurs actions disponibles.

Partager

La fonction Partager permet de gérer les options de partage d'un fichier ou d'un dossier. Cette section sera détaillée séparément afin de couvrir l'ensemble des possibilités liées aux droits et aux liens de partage.

Ajouter

Le menu Ajouter permet d'ajouter ou de créer du contenu dans l'emplacement sélectionné. En le déployant, les actions suivantes sont disponibles :

  • Importer un fichier
  • Importer un dossier
  • Créer un nouveau dossier
  • Créer une boîte de dépôt
  • Créer un document texte
  • Créer une feuille de calcul (tableau)
  • Créer une présentation
  • Créer un document texte simple
  • Créer une prise de notes
  • Créer un diagramme
  • Créer un raccourci web

Actions sur les fichiers et dossiers

Selon l'élément sélectionné (fichier ou dossier), plusieurs actions sont également disponibles :

  • Renommer
  • Télécharger
  • Demander des fichiers

Informations

La section Informations permet d'afficher les détails du fichier ou du dossier sélectionné.

Organiser

La fonction Organiser permet de gérer la structure et le positionnement des éléments dans l'arborescence.

Calculer la taille

Cette option permet de calculer la taille totale d'un dossier afin d'obtenir une estimation de l'espace occupé.

Placer à la corbeille

Cette action permet de supprimer un fichier ou un dossier en le plaçant dans la corbeille. Les éléments peuvent ensuite être restaurés selon les règles de conservation en vigueur.

3. Fonction Partager

Lors d'un clic droit sur un fichier ou un dossier, l'option Partager permet d'accéder aux paramètres de partage. Deux modes principaux sont disponibles.

1. Partager publiquement

Cette option permet de générer un lien de partage accessible à des personnes externes à l'organisation.

Activation du partage public
La première étape consiste à activer le partage public afin de générer un lien partageable.

Droits d'accès
Une fois le partage activé, il est possible de définir les droits associés :

  • Autoriser les modifications : permet aux personnes ayant le lien de modifier le document. Si cette option est désactivée, l'accès est limité à la lecture seule.

Paramètres de sécurité du lien

  • Autoriser le téléchargement : permet ou non aux utilisateurs de télécharger le fichier.
  • Date d'expiration : possibilité de définir une date et une heure à partir desquelles le lien ne sera plus valide.
  • Protection par mot de passe : ajoute une couche de sécurité supplémentaire. L'accès au document nécessite alors le lien + un mot de passe.
  • Autoriser les utilisateurs Infomaniak à demander l'accès : si activé, les utilisateurs disposant d'un compte Infomaniak peuvent faire une demande d'accès au fichier.

2. Utilisateurs avec accès

Cette section permet de gérer les partages internes avec des utilisateurs spécifiques déjà autorisés. Elle permet notamment de contrôler les personnes ou groupes ayant accès au fichier ou dossier, ainsi que leurs droits (lecture, modification, etc.).

📥

4. Boîte de dépôt

La boîte de dépôt est une fonctionnalité accessible depuis le menu Ajouter lors d'un clic droit dans un dossier. Elle permet de créer un espace dédié dans lequel des utilisateurs externes peuvent déposer des fichiers de manière simple et encadrée.

Création de la boîte de dépôt

Lors de la création, il est nécessaire de :

  • Donner un nom à la boîte de dépôt
  • Définir les paramètres d'accès et de fonctionnement

Paramètres disponibles

Plusieurs options peuvent être configurées :

  • Notification par e-mail : permet de recevoir un e-mail dès qu'un fichier est déposé
  • Date d'expiration : définit une date et une heure de fin de validité de la boîte
  • Protection par mot de passe : sécurise l'accès au lien de dépôt
  • Limitation de l'espace de stockage : permet de restreindre la taille totale des fichiers pouvant être déposés

Fonctionnement côté utilisateur

Une fois le lien de la boîte de dépôt transmis (par e-mail ou autre moyen), l'utilisateur accède à une page de dépôt. Sur cette page, il peut :

  • Cliquer sur le bouton "+" pour ajouter un fichier
  • Effectuer un glisser-déposer des documents
  • Renseigner ses informations (prénom, nom, adresse e-mail)

Résultat côté organisation

Une fois le dépôt effectué :

  • Un dossier est automatiquement créé avec le nom et le prénom de la personne
  • Les fichiers déposés sont stockés à l'intérieur de ce dossier

Usage

La boîte de dépôt fonctionne comme un canal de transfert simplifié et personnalisé, permettant de collecter des documents via un lien dédié. Elle constitue une alternative pratique aux outils de transfert classiques pour les utilisateurs moins familiers avec des solutions comme SwissTransfer.

👥

5. Collaboration en temps réel (partage public)

Lorsqu'un fichier est partagé via un lien de partage public, il est possible de travailler dessus simultanément avec d'autres utilisateurs, quel que soit le type de document (tableur, document texte ou présentation).

Accès via lien de partage

Une fois le lien transmis :

  • Le fichier s'ouvre directement depuis l'interface web
  • L'utilisateur accède à la version partagée du document

Édition collaborative en temps réel

Si les droits de modification sont activés :

  • Plusieurs personnes peuvent travailler simultanément sur le même fichier
  • Les modifications sont visibles en temps réel
  • Chaque utilisateur est identifié dans le document

Cas d'usage en réunion

Ce mode de collaboration est particulièrement utile lors de réunions à distance (ex. Teams ou autre outil de visioconférence) :

  • Tous les participants peuvent intervenir en même temps sur le même document
  • Les modifications sont visibles immédiatement
  • Le travail collaboratif est réalisé en direct, sans duplication de fichiers

Exemple

Un utilisateur peut collaborer simultanément avec une équipe sur un même fichier, par exemple un document de suivi, un tableau ou une présentation. Dans ce contexte, plusieurs collaborateurs travaillent en parallèle sur le même document, avec une synchronisation instantanée des modifications.

6. Éditeur en ligne kDrive et plugins (OnlyOffice)

L'éditeur en ligne intégré à kDrive repose sur la suite OnlyOffice, qui permet de créer et modifier des documents directement depuis le navigateur.

Fonctionnement général

L'éditeur permet de travailler sur différents types de fichiers (documents, tableurs, présentations) sans installation locale, avec des fonctionnalités collaboratives et d'édition en ligne.

Modules et extensions

OnlyOffice permet l'ajout de modules complémentaires (plugins) afin d'étendre ses fonctionnalités.

Plugin Draw.io

L'installation du plugin Draw.io permet d'intégrer la création de diagrammes directement dans les documents.

Utilisation

  • Ouvrir un document dans l'éditeur en ligne
  • Lancer le plugin Draw.io depuis l'interface OnlyOffice
  • Créer un diagramme (flux, schéma, organigramme, etc.)
  • Insérer le diagramme directement dans le document

Résultat

Une fois inséré :

  • Le diagramme est intégré au document Office
  • Il peut être repositionné et modifié selon les fonctionnalités disponibles
  • Il fait partie intégrante du fichier, sans export externe nécessaire
💻

7. Intégration kDrive et OnlyOffice (usage local)

En complément de l'éditeur en ligne kDrive, il est également possible d'utiliser OnlyOffice en local sur sa machine et d'y intégrer kDrive.

Principe

Lorsque OnlyOffice est installé sur le poste de travail, il est possible d'y connecter kDrive afin d'accéder directement aux fichiers stockés dans le cloud depuis l'application locale.

Fonctionnement

Une fois l'intégration configurée :

  • kDrive est accessible depuis l'interface OnlyOffice
  • Les fichiers stockés dans kDrive peuvent être ouverts directement dans OnlyOffice
  • Les documents peuvent être modifiés localement dans OnlyOffice
  • Les modifications sont ensuite synchronisées automatiquement avec kDrive

Avantages

Cette intégration permet :

  • De travailler dans un environnement local (OnlyOffice)
  • D'accéder directement aux fichiers kDrive sans passer par le navigateur
  • De bénéficier de la synchronisation automatique avec le cloud

Résultat

Les fichiers restent centralisés dans kDrive tout en étant éditables directement depuis OnlyOffice installé sur la machine, avec synchronisation des modifications.
